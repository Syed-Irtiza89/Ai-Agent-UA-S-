# Contributing to UA²S

1. **Fork** the repository.
2. **Clone** locally and create a new feature branch.
3. **Write unit tests** for any new agent or tool.
4. **Ensure Validator passes** for all changes.
5. **Submit a PR** with a clear explanation of your changes.

> All contributions must respect the 16-clause [CONSTITUTION.md](CONSTITUTION.md).
> Any PR that bypasses safety rules will be rejected.

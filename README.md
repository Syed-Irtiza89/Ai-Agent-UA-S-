# UA²S - Universal Autonomous AI Agent System

UA²S is a general-purpose autonomous AI agent system built with a focus on safety, autonomy, and long-term memory.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Run the system**:
    ```bash
    python main.py
    ```

## 🏗️ Architecture
- **Orchestration**: Graph-based state management (LangGraph-style).
- **Agents**: Multi-agent collaboration (CrewAI-style).
- **Safety**: 16-clause Agent Constitution enforced at runtime.
